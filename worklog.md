# MAJESTIC Motors - Luxury Car Rental Website

## Project Overview
A premium, $100K+ quality luxury car rental website built with Next.js 16, TypeScript, Tailwind CSS 4, Framer Motion, and shadcn/ui.

## Tech Stack
- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS 4 with custom dark/gold luxury theme
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Database**: Prisma ORM with SQLite
- **Fonts**: Playfair Display (display) + Geist (body)

## Features
- Full-screen cinematic hero with parallax particle effects
- AI-generated luxury car images (Ferrari, Lamborghini, Rolls-Royce, Porsche, Bentley, Mercedes-AMG)
- Filterable fleet showcase with animated cards
- Multi-step booking dialog with price calculator
- Brand timeline section
- Testimonials carousel with auto-play
- Services/why-choose-us section with animated icons
- Contact form with showroom image
- CTA banner footer with social links
- Glassmorphism navbar with scroll effect
- Gold shimmer text animations
- Custom scrollbar styling
- Fully responsive (mobile-first)
